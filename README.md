# BODY FEET Web

Sitio web comercial para BODY FEET, creado con Next.js 15, React, TypeScript, Tailwind CSS, Framer Motion y Lucide React.

## Datos usados del documento

- Nombre comercial: BODY FEET.
- Eslogan: "Tu recuperación es nuestra prioridad".
- Servicios: podología clínica y plantillas ortopédicas, terapia física y rehabilitación, quiropráctica, enfermería, cosmiatría y craneopuntura.
- Sedes: Flora Tristán, Musa y Manchay.
- Diferenciales: atención sin límite de tiempo, resultados desde la primera sesión, tecnología de vanguardia y técnicas complementarias.

## Pendientes del cliente

- Correo electrónico oficial.
- Enlaces exactos de Google Maps.
- Fotografías reales de sedes, equipo y servicios.
- Logotipo completo en SVG/PNG de alta resolución. El documento solo incluye isotipo y guía visual.

## Variables de entorno

```bash
NEXT_PUBLIC_SITE_URL=https://www.bodyfeet.pe
CONTACT_TO_EMAIL=correo-oficial@dominio.com
```

El endpoint `/api/contact` valida, sanitiza y registra la solicitud. Está preparado para integrarse con un servicio de correo, CRM o backend.
